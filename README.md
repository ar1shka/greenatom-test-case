## Настройка окружения

Обязательные:
```bash
export LOGIN=...
export PASSWORD=...
```

Необязательные (имеют значения по умолчанию):
```bash
export SERVER_ADDRESS="smtp.mail.ru"
export SERVER_PORT=465
```

## Запуск

```bash
python test.py
```
